# ✦ In the Light of Truth — The Grail Message

> *"The Message will strike only those who still carry within them a spark of truth and the yearning to be true human beings."* — Abd-ru-shin

## Overview

A full-featured spiritual bookstore website dedicated to the works of **Abd-ru-shin** (Oskar Ernst Bernhardt, 1875–1941), author of *In the Light of Truth – The Grail Message*.

Built for earnest seekers across Kenya and Africa, this website enables the discovery, purchase, and audio enjoyment of Abd-ru-shin's complete works — with pricing in Kenya Shillings (KSH) and M-Pesa payment integration.

---

## Features

- 📖 **Author Biography** — Full history of Abd-ru-shin with timeline
- 📚 **Bookstore** — 6 titles with descriptions and KSH pricing
- 🛒 **Shopping Cart** — Add/remove items, quantity control, cart total
- 💳 **Payment Options** — M-Pesa (Lipa Na M-Pesa), Visa/Mastercard, Bank Transfer
- 🎧 **Audio Library** — Chapter-by-chapter audio with download-for-fee system
- 📥 **Free Sample Download** — Gated behind a full sign-up form
- 🌌 **Core Teachings** — 8 thematic panels explaining the Grail Message
- 📱 **Fully Responsive** — Works on mobile, tablet, desktop

## Deployment on GitHub Pages

```bash
# 1. Clone this repository
git clone https://github.com/molich023/grailmessage.git

# 2. Navigate into the folder
cd grailmessage

# 3. Push to GitHub
git add .
git commit -m "Initial commit — Grail Message website"
git push origin main
```

Then in your GitHub repository:
- Go to **Settings → Pages**
- Set source to **main branch / root**
- Your site will be live at: `https://molich023.github.io/grailmessage`

---

## M-Pesa Integration (Next Steps)

To enable live M-Pesa payments, you will need:
1. A **Safaricom Daraja 2.0** developer account
2. A **Node.js/Express** backend (can be deployed on Render.com)
3. Paybill or Buy Goods number from Safaricom

The frontend cart is ready to connect — just add your Daraja API endpoint in the checkout handler.
