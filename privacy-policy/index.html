Section 4

Data Storage and Security

Your data is stored securely using Supabase — a PostgreSQL database hosted on AWS infrastructure with encryption at rest and in transit. All connections use SSL/TLS encryption.

Our website is hosted on Netlify with HTTPS enforced, security headers, and DDoS protection. Access to your data is protected by Row Level Security policies — no other user can access your personal data.

Your passwords are never stored in plain text. They are hashed and encrypted by Supabase Auth before storage. Even we cannot see your password.

Security measures in place: SSL encryption · Password hashing · Row Level Security · Rate limiting · Bot protection · Input sanitization · Security headers (CSP, HSTS, X-Frame-Options)
